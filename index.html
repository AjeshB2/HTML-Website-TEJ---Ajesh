PassLock is a device created using the Microbit and a servo motor to create a password lock system. Problem Adressed

My device will solve the problem of home security. This helps people make sure only the residents of a home are going into it. The password lock prevents criminals from potentially breaking into a house.

Target

PassLock targets any person who values the safety of their home. This also allows loved ones to have access to their home while preventing unwanted people from coming in.

Core Features

The PassLock features a central microbit that acts as the input for the password. This will be done by using the A and B buttons on the microbit which will be put in a certain combination to unlock the door. The door will open using a servo motor which adjusts the angle of the door to open. If the password is deemed incorrect, the door will stay shut and the microbit plays an alarm noise.
